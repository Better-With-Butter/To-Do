Download the .exe or build it yourself using python. On first boot, it will ask you what folder you want to save the file to.

You will need to install tkcalendar for this to work:

```
pip install tkcalendar
```

You can double click the checkbox icon to change it to in progress, completed, or canceled.

If you are saving the file to your Obsidian Vault (Or other markdown reader) it will update live!
